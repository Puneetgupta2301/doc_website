# Dr. Pradeep Agrawal – Premium Doctor Portfolio

## Files
- `index.html` — Main page (all sections)
- `style.css` — Complete styling (CSS Variables, Flexbox, Grid)
- `script.js` — All JS interactivity (vanilla, no dependencies)
- `assets/` — Place your images here (see below)

## Assets Required
Place these images in the `/assets/` folder:

| File | Usage | Recommended Size |
|------|-------|------------------|
| `doctor-hero.jpg` | Hero section portrait | 480×560 px |
| `about-doctor.jpg` | About section portrait | 440×540 px |
| `favicon.ico` | Browser tab icon | 32×32 px |

If images are not provided, placeholder images are shown automatically.

## Deploy on GitHub Pages
1. Create a new GitHub repository
2. Upload `index.html`, `style.css`, `script.js`, and the `assets/` folder
3. Go to Settings → Pages → Source: Deploy from branch `main`, folder `/root`
4. Your site will be live at `https://yourusername.github.io/repo-name/`

## Features
- Sticky responsive navbar with hamburger menu
- Full-page hero with doctor photo and CTAs
- Animated counters (20+, 10000+, 4.9★, 95%)
- Services grid with hover effects
- Why Choose section
